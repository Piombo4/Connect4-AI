# Connect4 AI
C++ implementation of Connect 4 using Minimax with Alpha-beta pruning.

## Compilation and Execution
Compilation:
</br>
</br>
 ```$ g++ main.cpp -o <name>```
</br>
</br>
Execution:
</br>
</br>
  ```$ ./<name> <depth>```
</br>
</br>
Where ```<depth>``` specifies the depth for minimax.
