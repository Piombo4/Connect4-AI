# Connect4 AI
C++ implementation of Connect 4 using Minimax with Alpha-beta pruning.

## Compilation and Execution
Compilation:
 ```$ g++ main.cpp -o <name>```
Execution:
  ```$ ./<name> <depth>```
<depth> specifies the depth for minimax.
